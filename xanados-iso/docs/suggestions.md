# Optional Upgrades & Enhancements for XanadOS

🎨 UI / Visual Experience
Feature Description
🖼️ Animated Wallpapers (via Komorebi or Plasma plugins)
Add high-quality animated backgrounds

Feature Description
🔧 GUI Kernel Manager Simple tool for switching kernels post-install

🧠 Power Profile Daemon power-profiles-daemon or auto-switching based on usage
🌐 VPN Toggle GUI Easy toggle for common VPN configs (Mullvad, ProtonVPN, etc.)
📦 AUR Watcher Tray icon to notify of AUR package updates
🛡️ Security Dashboard Show sandboxed app states, firewall status, AppArmor events

🎮 Steam Tweaks Panel UI for toggling Proton GE, shader prefetch, etc.

📊 FPS + Performance Overlay Toggle GUI switch for MangoHud profiles
🔄 Wine Manager GUI Tool for managing Wine versions, prefixes, DXVK overrides
